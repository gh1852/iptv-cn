# iptv-cn
自用国内直播频道（定期更新～）

# 使用方式
![](.readme/vlc.png)
打开支持m3u格式软件输入网络URL`https://raw.fastgit.org/gh1852/iptv-cn/master/iptv-cn.m3u`

# [支持m3u格式软件参考awesome-iptv](https://github.com/iptv-org/awesome-iptv#apps)
